LOGO INSTRUCTIONS
================

1. Replace this file with your actual logo image
2. Name your logo file: "logo.png"
3. Recommended size: 60x60 pixels or larger (will be scaled down)
4. Supported formats: PNG, JPG, JPEG
5. Place the logo file in this Images folder

The logo will appear at the top of every bill receipt, centered above the store name.

For best results:
- Use a PNG file with transparent background
- Keep the file size under 100KB
- Use a square aspect ratio (1:1)
- Ensure good contrast for thermal printing
